Hot keys

  • 'F1' - Help
  • Изображения пролистываются кнопками "лево"-"право", сохраняя координаты bboxes последней картинки
  • 'P' - переносит координаты bboxes из памяти на текущее изображение
  • 'Q' - рисует горизонтальные линии по левым грамям bboxes
  • 'W' - рисует горизонтальные линии по правым грамям bboxes
  • 'R', 'Del', 'Backspace' - удаляют выбранный bbox
  • 'C' - удаляет все bboxes с текущего изображения
  • 'Z' - заливает bboxes одним цветом
  • 'X' - заливает все изображение вокруг bboxes одним цветом
  • При зажатой 'стрелка вверх' - заливает bboxes одним цветом
  • При зажатой 'стрелка вниз' - заливает все изображение вокруг bboxes одним цветом
